La mise en oeuvre de l'application necessite l'utilisation d'IntelliJ

Il faut disposer de la bibliotheque javafx.

Il faut :
- creer un projet maven
- recopier tous le dossier src
- remplacer le fichier pom.xml a la racine du projet par le nouveau fichier pom.xml
- copier le fichier iris.csv a la racine du projet

Ensuite il faut recharger le projet maven :
- clic droit sur le dossier du projet
- descendre a maven
- choisir reload project


Le main du projet est situe dans src/main/java/main/Main.java
